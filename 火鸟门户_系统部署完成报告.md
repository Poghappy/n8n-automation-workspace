## 🎯 使用指南

### 启动系统

1. **启动N8N服务**:
   ```bash
   npx n8n start
   ```

2. **启动新闻API服务**:
   ```bash
   source venv/bin/activate
   python -c "
   import uvicorn
   from src.api.news_api import app
   uvicorn.run(app, host='0.0.0.0', port=3001)
   "
   ```

3. **访问N8N界面**: http://localhost:5678
